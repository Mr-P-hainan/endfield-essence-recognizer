# 终末地基质妙妙小工具

不知道哪些基质该留哪些该扔？逝逝这个妙妙小工具吧！

## 使用前阅读
  - 请打开终末地贵重品库并切换到武器基质页面
  - 在运行过程中，请确保终末地窗口置于前台

## 功能介绍
  - 按 "[" 键识别当前基质，仅识别不操作
  - 按 "]" 键扫描所有基质，并自动锁定宝藏基质，解锁垃圾基质
    基质扫描过程中再次按 "]" 键中断扫描
  - 按 "Alt+Delete" 退出程序
